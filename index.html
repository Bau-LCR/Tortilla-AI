📋 ¿Cómo configurar las 5 API Keys?

  1. Entrá a console.groq.com/keys y creá 5 API Keys.
  2. En Vercel, andá a tu proyecto → Settings → Environment Variables.
  3. Agregá las siguientes variables con cada key respectiva:
GROQ_API_KEY→ Tu primera API Key (ya existente)
GROQ_API_KEY_2→ Segunda key
GROQ_API_KEY_3→ Tercera key
GROQ_API_KEY_4→ Cuarta key
GROQ_API_KEY_5→ Quinta key

⚡ El servidor rota automáticamente cuando una key recibe un error 429 (rate limit). Con 5 keys de 10.000 tokens cada una = 50.000 tokens antes del bloqueo total.
